1.Builder Pattern

  • Pada program ini, class ConfigBuilder digunakan untuk menerapkan konsep builder pattern dalam pembuatan objek Config.
  • Method host() dan port() digunakan untuk mengatur nilai konfigurasi secara bertahap sebelum objek dibuat.
  • Setiap method pada class ConfigBuilder mengembalikan nilai this sehingga method dapat dipanggil secara berantai (method chaining).
  • Method build() digunakan untuk membuat dan mengembalikan objek Config setelah semua konfigurasi selesai diatur.
  • Pada fungsi main(), objek konfigurasi dibuat dengan cara memanggil method secara berurutan menggunakan konsep builder pattern.
  • Program kemudian menampilkan informasi host, port, dan pesan bahwa konfigurasi berhasil dibuat.

Kode Dart:


// ==========================================
// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A
// ==========================================

class Config {}

class ConfigBuilder {

  ConfigBuilder host(String h) {
    print("Host : $h");
    return this;
  }

  ConfigBuilder port(int p) {
    print("Port : $p");
    return this;
  }

  Config build() {
    print("Config berhasil dibuat");
    return Config();
  }
}

void main() {

  ConfigBuilder()
      .host('localhost')
      .port(8080)
      .build();
}

2.Masalah Tanpa Extension Methods

  • Pada program ini, class StringUtils digunakan sebagai class bantuan yang berisi method statis untuk memproses data bertipe string.
  • Method capitalize() digunakan untuk mengubah huruf pertama pada teks menjadi huruf kapital.
  • Method isEmail() digunakan untuk memeriksa apakah suatu teks mengandung karakter @ sebagai indikator format email.
  • Pemanggilan method dilakukan melalui nama class StringUtils sehingga kode terlihat lebih panjang dan kurang praktis.
  • Pada fungsi main(), program melakukan proses kapitalisasi nama dan pengecekan email menggunakan method statis.
  • Program kemudian menampilkan hasil pengolahan string ke layar.

Kode Dart:

class StringUtils {

  static String capitalize(String text) {
    if (text.isEmpty) return text;
    return text[0].toUpperCase() + text.substring(1);
  }

  static bool isEmail(String text) {
    return text.contains('@');
  }
}

void main() {

  var name = 'john';

  var capitalized = StringUtils.capitalize(name);
  var isEmail = StringUtils.isEmail('test@mail.com');

  print('Original name : $name');
  print('Capitalized    : $capitalized');
  print('Is Email       : $isEmail');
}

3.Solusi dengan Extension Methods

  • Pada program ini, extension methods digunakan untuk menambahkan method baru pada tipe data String tanpa mengubah class aslinya.
  • Method capitalize() digunakan untuk mengubah huruf pertama menjadi huruf kapital.
  • Getter isEmail digunakan untuk memeriksa apakah teks memiliki format email sederhana.
  • Method reverse() digunakan untuk membalik urutan karakter pada string.
  • Dengan menggunakan extension methods, pemanggilan method menjadi lebih singkat, rapi, dan mudah dibaca karena dapat langsung dipanggil dari objek string.
  • Program kemudian menampilkan hasil kapitalisasi, pengecekan email, dan pembalikan teks ke layar.

Kode Dart:


// ==========================================
// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A
// ==========================================

extension StringExtensions on String {

  String capitalize() {
    if (isEmpty) return this;
    return this[0].toUpperCase() + substring(1);
  }

  bool get isEmail => contains('@');

  String reverse() {
    return split('').reversed.join();
  }
}

void main() {

  var name = 'john';

  var capitalized = name.capitalize();
  var isEmail = 'test@mail.com'.isEmail;
  var reversed = 'hello'.reverse();

  print('Original name : $name');
  print('Capitalized    : $capitalized');
  print('Is Email       : $isEmail');
  print('Reverse hello  : $reversed');
}

4.Sintaks Extension Methods

  • Pada program ini, extension methods digunakan untuk menambahkan method baru pada tipe data DateTime.
  • Method toFormattedString() digunakan untuk mengubah format tanggal menjadi lebih mudah dibaca.
  • Getter isToday digunakan untuk memeriksa apakah tanggal yang digunakan sama dengan tanggal hari ini.
  • Operator + digunakan untuk menambahkan durasi waktu pada objek DateTime.
  • Pada fungsi main(), program menampilkan tanggal saat ini, memeriksa apakah tanggal tersebut adalah hari ini, dan menambahkan satu hari ke tanggal sekarang.
  • Program kemudian menampilkan hasil pengolahan tanggal ke layar.

Kode Dart:


// ==========================================
// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A
// ==========================================

extension DateTimeExtensions on DateTime {

  // Method
  String toFormattedString() {
    return '$day/$month/$year';
  }

  // Getter
  bool get isToday {
    var now = DateTime.now();

    return year == now.year &&
           month == now.month &&
           day == now.day;
  }

  // Operator
  DateTime operator +(Duration duration) {
    return add(duration);
  }
}

void main() {

  DateTime today = DateTime.now();

  print('Tanggal hari ini        : ${today.toFormattedString()}');
  print('Apakah hari ini?        : ${today.isToday}');

  DateTime nextDay = today + Duration(days: 1);

  print('Tanggal setelah ditambah: ${nextDay.toFormattedString()}');
}

6.Extension untuk Null Safety

  • Pada program ini, extension methods digunakan pada tipe data String? untuk mendukung konsep null safety.
  • Method orEmpty() digunakan untuk mengubah nilai null menjadi string kosong.
  • Getter isNullOrEmpty digunakan untuk memeriksa apakah string bernilai null atau kosong.
  • Getter isNotNullOrEmpty digunakan untuk memeriksa apakah string memiliki isi.
  • Method withDefault() digunakan untuk memberikan nilai default jika string bernilai null.
  • Program kemudian menampilkan hasil penggunaan extension pada nullable string ke layar.

Kode Dart:


// ==========================================
// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A
// ==========================================

extension NullableStringExtensions on String? {

  String orEmpty() => this ?? '';

  bool get isNullOrEmpty => this == null || this!.isEmpty;

  bool get isNotNullOrEmpty => !isNullOrEmpty;

  String withDefault(String defaultValue) {
    return this ?? defaultValue;
  }
}

void main() {

  String? name;

  print('Hasil orEmpty()            : ${name.orEmpty()}');
  print('Hasil isNullOrEmpty        : ${name.isNullOrEmpty}');
  print('Hasil isNotNullOrEmpty     : ${name.isNotNullOrEmpty}');
  print('Hasil withDefault("Guest") : ${name.withDefault('Guest')}');
}

7.Praktik 1 – Utilities untuk Dart Types

  • Pada program ini, extension methods digunakan pada tipe data num untuk menambahkan beberapa fungsi tambahan.
  • Method toRupiah() digunakan untuk mengubah angka menjadi format mata uang rupiah.
  • Method isBetween() digunakan untuk memeriksa apakah suatu angka berada di antara nilai minimum dan maksimum.
  • Method power() digunakan untuk menghitung nilai perpangkatan dari sebuah angka.
  • Pada fungsi main(), program melakukan format angka ke rupiah, pengecekan range angka, dan perhitungan perpangkatan.
  • Program kemudian menampilkan hasil pengolahan angka ke layar.

Kode Dart:


// ==========================================
// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A
// ==========================================

extension NumberExtensions on num {

  // Format ke Rupiah
  String toRupiah() {
    return 'Rp${toStringAsFixed(0)}';
  }

  // Cek nilai dalam range
  bool isBetween(num min, num max) {
    return this >= min && this <= max;
  }

  // Pangkat
  num power(int exponent) {

    var result = this;

    for (var i = 1; i < exponent; i++) {
      result *= this;
    }

    return result;
  }
}

void main() {

  var price = 150000;

  print('Harga dalam rupiah : ${price.toRupiah()}');
  print('Apakah antara 1-10 : ${5.isBetween(1, 10)}');
  print('Hasil 2 pangkat 3  : ${2.power(3)}');
}
