1.Builder Pattern

  • Pada program ini, Bentuk digunakan sebagai abstract class yang berfungsi sebagai dasar untuk mendefinisikan berbagai jenis bangun datar melalui method hitungLuas().
  • Class Bentuk tidak dapat dibuat objek secara langsung karena bersifat abstrak dan hanya digunakan sebagai kerangka dasar bagi class turunannya.
  • Class Persegi, Lingkaran, dan Segitiga merupakan turunan dari class Bentuk yang mengimplementasikan method hitungLuas() sesuai dengan rumus masing-masing bangun.
  • Setiap class turunan memiliki cara perhitungan luas yang berbeda, sehingga konsep polimorfisme dapat diterapkan pada program ini.
  • Pada fungsi main(), objek dari setiap class dibuat untuk melakukan proses perhitungan luas berdasarkan data yang diberikan.
  • Program kemudian menampilkan hasil perhitungan luas dari bangun persegi, lingkaran, dan segitiga ke layar sebagai output akhir.

Kode Dart:


// ==========================================

class Config {}

class ConfigBuilder {

  ConfigBuilder host(String h) {
    print("Host : $h");
    return this;
  }

  ConfigBuilder port(int p) {
    print("Port : $p");
    return this;
  }

  Config build() {
    print("Config berhasil dibuat");
    return Config();
  }
}

void main() {

  ConfigBuilder()
      .host('localhost')
      .port(8080)
      .build();
}
