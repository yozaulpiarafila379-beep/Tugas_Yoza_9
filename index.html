1.Builder Pattern

  • Pada program ini, ConfigBuilder digunakan untuk menerapkan konsep Builder Pattern dalam pembuatan objek Config.
  • Method host() dan port() digunakan untuk mengatur nilai konfigurasi secara bertahap sebelum objek dibuat.
  • Setiap method pada ConfigBuilder mengembalikan nilai this sehingga method dapat dipanggil secara berantai (method chaining).
  • Method build() digunakan untuk membuat dan mengembalikan objek Config setelah semua konfigurasi selesai diatur.
  • Pada fungsi main(), objek konfigurasi dibuat dengan cara memanggil method secara berurutan menggunakan konsep Builder Pattern.
  • Program kemudian menampilkan informasi host, port, dan pesan bahwa konfigurasi berhasil dibuat.

Kode Dart:


// ==========================================
// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A
// ==========================================

class Config {}

class ConfigBuilder {

  ConfigBuilder host(String h) {
    print("Host : $h");
    return this;
  }

  ConfigBuilder port(int p) {
    print("Port : $p");
    return this;
  }

  Config build() {
    print("Config berhasil dibuat");
    return Config();
  }
}

void main() {

  ConfigBuilder()
      .host('localhost')
      .port(8080)
      .build();
}

2.Masalah Tanpa Extension Methods

  • Pada program ini, StringUtils digunakan sebagai class bantuan yang berisi method statis untuk memproses data bertipe string.
  • Method capitalize() digunakan untuk mengubah huruf pertama pada teks menjadi huruf kapital.
  • Method isEmail() digunakan untuk memeriksa apakah suatu teks mengandung karakter @ sebagai indikator format email.
  • Pemanggilan method dilakukan melalui nama class StringUtils sehingga kode terlihat lebih panjang dan kurang praktis.
  • Pada fungsi main(), program melakukan proses kapitalisasi nama dan pengecekan email menggunakan method statis.
  • Program kemudian menampilkan hasil pengolahan string ke layar.

Kode Dart:


// ==========================================
// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A
// ==========================================

class StringUtils {

  static String capitalize(String text) {
    if (text.isEmpty) return text;
    return text[0].toUpperCase() + text.substring(1);
  }

  static bool isEmail(String text) {
    return text.contains('@');
  }
}

void main() {

  var name = 'john';

  var capitalized = StringUtils.capitalize(name);
  var isEmail = StringUtils.isEmail('test@mail.com');

  print('Original name : $name');
  print('Capitalized    : $capitalized');
  print('Is Email       : $isEmail');
}
